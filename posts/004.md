# LYNX

Hi again - it's me.

Lynx is an internet browser as old as time. And I hope it stays alive forever more. It only has one flaw, which barely is a flaw. But I'll get to that later.

First things first - this post will tell you how **I** use Lynx. You can use it however you like, or not at all if that's your jam.

Second things, **this** is how I use it. I don't really change anything in the lynx.cfg, except these two thing:

1. Turn the VI-keys on. Arrow keys are for children and beast-prople.
2. Make sure you add **TEXTFIELDS_NEED_ACTIVATION:TRUE** to your lynx.cfg. You can find the config file in _/etc/lynx.cfg_ (most of the time at least). This makes it so that when you're scrolling down a page and come across a text input field, it doesn't automatically stop and go into edit mode. You now have to select the text field in which you want to input text.
3. BONUS!! Do yourself a favour and put _~/lynx_bookmarks.html_ as your start page. This makes for a flexible and fast way to get started when you open up the browser.

So what's the catch with using Lynx? Well, the biggest flaw, if you wanna call it that, is that it cannot render JavaScript. This means that, ironically, this site cannot be viewed is Lynx! It just looks like this:

[![](/assets/lynx02.png "this site in lynx")](/assets/lynx02.png)

It doesn't display any of my posts, beacuse they are rendered in JavaScript. This is true for alot of modern web pages out there, but most news sites, forums, and small web pages work great in Lynx.

But... Why on earth would you want to see the internet as just... Text? WELL!!:

- I believe that scaling down the number of impressions you get when you go to a web page makes you keep your focus on why even entered the page in the first place.
- No ads
- Again, no ads
- It is, for the most time, crazy fast
- It looks really fucking cool. I mean look at this:

[![](/assets/lynx03.png "swedish news website in lynx")](/assets/lynx03.png)

It's crazy dope.

Last things last, I make a little shell script that lets me enter something like

```bash
? a question to be searched on the web
```

into my terminal. It then searches for the string on duckduckgo lite in Lynx. Here it is, if you want it.

```bash
#!/bin/bash

query="$*"
if [ -z "$query"]; then
  echo "Användning: ? sökfras"
  exit 1
fi

lynx "https://lite.duckduckgo.com/lite/?q=$(printf '%s' "$query" | sed 's/ /+/g')"
```
